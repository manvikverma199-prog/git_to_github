1. Create a github account.
2. Create a project folder/repo in local computer.
3. Install git.
4. Go to the folder location (inside folder) and right click and open git bash here.
5. write git init
6. git add .
7. git config --global user.name "your username" (any username can be given).
8. git config --global user.email "your email" (same as github)
9. git commit -m "initial commit"
10. git remote add origin https://github.com/yoururname/gitreponame.git (get this url from github)
11. git remote -v (check status)
12. git brnch -M main
13. git push -u origin main (push code, it will ask to login to git for authentication)
